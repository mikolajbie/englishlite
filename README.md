# Lite English

> Experimental simplified English grammar system

Lite English is a minimal grammar layer on top of standard English.  
It removes irregular verb forms and reduces tense complexity while keeping vocabulary and spelling unchanged.

It is not a replacement for English, but a structured simplification model for communication.

---

## Core Principles

- One verb form only (base form)
- No irregular verbs
- No subject-based conjugation
- Tense is expressed using auxiliary operators
- Negation is standardized

---

## Pronouns

- I
- you
- we
- they (covers he / she / it)

---

## Verb Rule

All verbs use base form:

- go
- eat
- see
- work

No conjugation.

---

## Tenses

### Present

I go
they eat
I don’t go


### Past

I did go
they did see
I didn’t go


### Future

I will go / I’ll go
they will eat
I won’t go


### Conditional

I would go
they would see
I wouldn’t go


---

## Negation Rules

- don’t → present
- didn’t → past
- won’t → future
- wouldn’t → conditional

Same rule for all subjects.

---

## Example

Standard:
I went to school because I needed to learn English.

Lite:
I did go to school because I need learn English.
